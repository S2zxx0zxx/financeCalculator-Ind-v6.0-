## What is FinCalc India?

A **single-file (~85KB) finance calculator suite** built for every Indian — from a ₹1,000 two-wheeler loan to a ₹10 Crore commercial property. No signup, no server, no tracking. Open the page, calculate, done.

Plus a growing **personal finance blog** with expert guides on EMI, SIP, Tax, GST — written in simple Hinglish for real Indians. And now a full **podcast page** with bite-sized audio finance guides coming soon.

> Built by a solo Indian developer using nothing but vanilla HTML, CSS, and JavaScript. Zero npm. Zero webpack. Zero frameworks. Just one file that works everywhere.

**Live at** → [satzzxzxx.me](https://satzzxzxx.me)

## Quick Start

**Just want to use it?** → Go to [satzzxzxx.me](https://satzzxzxx.me)

**Run locally:**

```bash
git clone https://github.com/S2zxx0zxx/financeCalculator-Ind-v6.0-.git
cd financeCalculator-Ind-v6.0-

# Just open index.html in any browser.
# No npm install. No build step. No server needed.
open index.html
```

**Deploy your own fork:**

```bash
# 1. Fork this repo on GitHub
# 2. Settings → Pages → Source: Deploy from main branch
# 3. Your site is live at:
#    https://YOUR_USERNAME.github.io/financeCalculator-Ind-v6.0-/

# Optional: Add custom domain in Settings → Pages → Custom Domain
```

**Add a new calculator:**

```bash
# All calculators live in index.html under the <script> section
# Each one follows this pattern:
#   - Input HTML block with unique IDs
#   - calcXXX() function with formula
#   - renderXXXChart() function using Canvas API
#   - Result display with animation + WhatsApp share
```
