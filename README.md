## 🔥 The Problem We Solve

<table>
<tr>
<td width="50%">

### ❌ Before FinCalc India

- Finance calculators are scattered across **50+ different websites**
- Most are **loaded with intrusive ads** that make them unusable

- Built for **₹1,000 to ₹10 Crore** — rickshaw driver to CEO
- **Dark/Light mode + Hindi/English** — fully accessible

## 🧮 Calculator Deep Dive

<details>
<summary><b>🏠 EMI Calculator</b> — Home Loan, Car Loan, Personal Loan</summary>

<br/>

| Feature | Details |
|---------|---------|
| **Formula** | `EMI = P × r × (1+r)^n / ((1+r)^n - 1)` |
| **Range** | ₹1,000 → ₹10,00,00,000 (₹10 Crore) |
| **Interest Rate** | 1% → 36% p.a. |
| **Tenure** | 1 → 30 Years |
| **Output** | Monthly EMI, Total Interest, Total Payable, Principal vs Interest Ratio |
| **Visual** | Animated Donut Chart (Canvas API) |
| **Actions** | Share (WhatsApp/Telegram), PDF Download, Save to History |

**Use Case:** A daily wage worker taking a ₹50,000 two-wheeler loan at 10% for 3 years sees ₹1,613/month EMI instantly. A businessman checking ₹5 Crore commercial property loan gets the same precision.

</details>

<details>
<summary><b>📈 SIP Returns Calculator</b> — Mutual Fund Wealth Builder</summary>

<br/>

| Feature | Details |
|---------|---------|
| **Formula** | `FV = P × ((1+r)^n - 1) / r × (1+r)` |
| **Monthly SIP** | ₹100 → ₹10,00,000 |
| **Expected Return** | 1% → 30% p.a. |
| **Period** | 1 → 40 Years |
| **Output** | Maturity Value, Amount Invested, Wealth Gained, ROI % |

**Use Case:** A college student investing ₹500/month at 12% for 5 years sees ₹41,243. An HNI investing ₹5L/month for 20 years sees ₹4.99 Crore.

</details>

<details>
<summary><b>📋 Income Tax Calculator 2025-26</b> — New vs Old Regime</summary>

<br/>

| Feature | Details |
|---------|---------|
| **New Regime** | Updated 2025-26 slabs with ₹75K standard deduction |
| **Old Regime** | With 80C, 80D, HRA deductions |
| **Rebate 87A** | Auto-applied for income ≤ ₹12L (New Regime) |
| **Age Groups** | Below 60, Senior (60-79), Super Senior (80+) |
| **Output** | Side-by-side comparison, Slab-wise breakup table, Recommendation |

**Use Case:** A ₹3L/year worker sees zero tax under both regimes. A ₹25L/year professional sees exactly which regime saves more, with rupee-by-rupee breakup.

</details>

<details>
<summary><b>🧾 GST Calculator</b> — Add/Remove GST with CGST+SGST Split</summary>

<br/>

| Feature | Details |
|---------|---------|
| **Modes** | Add GST (Exclusive) / Remove GST (Inclusive) |
| **Rates** | 5%, 12%, 18%, 28% |
| **Output** | Base Amount, Total GST, CGST (50%), SGST (50%), IGST info |

</details>

<details>
<summary><b>🏦 FD Calculator</b> — Fixed Deposit Maturity</summary>

<br/>

| Feature | Details |
|---------|---------|
| **Compounding** | Quarterly, Monthly, Half-Yearly, Yearly |
| **Deposit Range** | ₹1,000 → ₹10 Crore |
| **Tenure** | 1 → 30 Years |
| **Visual** | Animated Donut (Deposit vs Interest) |

</details>

<details>
<summary><b>💰 RD + 🏖️ Retirement + 📊 Inflation + ✅ Loan Eligibility + 🏡 Rent vs Buy</b></summary>

<br/>

All calculators follow the same design philosophy:
- **Low defaults** for accessibility (₹500 RD, ₹5,000 monthly expense for retirement)
- **Wide ranges** up to ₹10 Crore for high-net-worth calculations
- **Live calculation** on every input change (debounced 200ms)
- **Animated results** with smooth reveal transitions
- **Share + Save + PDF** actions on every calculator

</details>

## 🚀 Quick Start

### Option 1: Direct Open (Zero Setup)

```bash
# Just download and open
curl -O https://raw.githubusercontent.com/Satzzxzxx/fincalc-india/main/FinCalc.html
open FinCalc.html    # macOS
xdg-open FinCalc.html  # Linux
start FinCalc.html   # Windows
```

### Option 2: GitHub Pages (Free Hosting)

```bash
# Clone → Rename → Deploy
git clone https://github.com/Satzzxzxx/fincalc-india.git
cd fincalc-india
mv FinCalc.html index.html
git add . && git commit -m "deploy" && git push

# Then: Settings → Pages → Deploy from main branch
# Live at: https://satzzxzxx.github.io/fincalc-india/
```

### Option 3: Custom Domain (Production)

```bash
# With Namecheap domain + GitHub Pages
echo "fincalcindia.com" > CNAME
git add CNAME && git commit -m "custom domain" && git push

# Namecheap DNS Settings:
# Type: CNAME | Host: www | Value: satzzxzxx.github.io
# Type: A    | Host: @   | Value: 185.199.108.153
```
